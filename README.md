# Tokin Market

Мини-проект маркетплейса на HTML/CSS/JS + Firebase для GitHub Pages.

## Что внутри
- вход через Google
- анонимный вход
- Firestore: пользователи, товары, покупки, отзывы
- Storage: фото товаров
- покупка за токены
- публикация товара: +20 токенов
- отзыв после покупки: +10 токенов
- страница магазина продавца
- тёмная glass-тема

## Файлы
- `index.html`
- `style.css`
- `app.js`
- `firebase.js`

## Как запустить
1. Создай Firebase project.
2. Включи **Authentication**:
   - Google sign-in
   - Anonymous sign-in
3. Включи **Firestore** и **Storage**.
4. В `firebase.js` вставь свой `firebaseConfig`.
5. Залей файлы в GitHub repository.
6. В GitHub Pages выбери публикацию из корня репозитория.

## Важно
Без Cloud Functions правило **«через 12 часов автоматически»** сделано на клиенте:
- сайт сам проверяет просроченные отзывы при входе и периодически во время работы;
- если никто не откроет сайт, награда выдастся при следующем заходе.
